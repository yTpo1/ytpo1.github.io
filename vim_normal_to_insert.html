Vim. Switching from Normal Mode to Insertion Mode. Quick Movements.

  • Esc A: go to the end of the line and enter Insertion Mode
  • Esc I: go to the start of the line and enter Insertion Mode
  • Esc o: start editing on a new line below the current line
  • Esc O: start editing on a new line above the current line
