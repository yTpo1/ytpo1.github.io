# VS Code

https://code.visualstudio.com/docs/getstarted/tips-and-tricks

* `Ctrl + P` go to file
* `Ctrl + Shift + p` open command pallet
* `Ctrl+,` open settings
* Color Theme command `(Ctrl+K Ctrl+T)` to display the Color Theme picker.
