grep

Useful alias set up

.zalias

  • -I: ignore binary files
  • --color=auto: add the color formatting
alias grep='grep --color=auto --exclude-dir={.bzr,CVS,.git,.hg,.svn,venv} --exclude=\*.{o,png,jpg,jpeg,gif,webp,mp3,flac,pdf,xcf} -I'

Examples

Search only the current directory:

grep “string*” *
