vim normal mode hotkeys

Navigation tip. All of these commands are easy to remember with some simple mnemonics: up down forward back Home Middle Last src

Navigation:

  • ctrl+u, ctrl+d: Moves cursor & screen Up/Down half a page
  • H, M, L: Go to Top/Middle/Bottom of the screen
  • ctrl+y, ctrl+e: Move screen Up/Down with one line
  • ), (: Line forward/backward
  • }, {: Paragraph forward/backward
  • ]m, [m: Go to next/previous start of method
  • ]M, [M:Go to next/previous end of method

Editing:

  • YP: Duplicate line

Other:

  • ctrl+l: Redraw screen
