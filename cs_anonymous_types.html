Anonymous Objects. Anonymous Types

In C#, anonymous types are created using the new keyword followed by an object initializer.

Links:

  • https://learn.microsoft.com/en-us/dotnet/csharp/fundamentals/types/anonymous-types

Example:

var person = new
{
    FirstName = "Alice",
    LastName = "Smith",
    Age = 30
};

Array of anonymously typed elements

var anonArray = new[] {
    new { name = "apple", diam = 4 },
    new { name = "grape", diam = 1 }
};
