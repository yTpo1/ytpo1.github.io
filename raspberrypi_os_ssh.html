Raspberry Pi OS. Enabling SSH

Preliminary step:

  1. Ensure SSH is Enabled on Raspberry Pi
    • Add an empty file named ssh (without any extension) to the boot partition of the Raspberry Pi's SD card. You can do this on your laptop before inserting the SD card into the Raspberry Pi.
    • Alternatively, if you have access to the Raspberry Pi's desktop, you can enable SSH through raspi-config: sudo raspi-config. Navigate to Interfacing Options > SSH and enable it.
